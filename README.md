# JGSK_EDU 강의 자료실

온라인 강의 플랫폼입니다.

## 파일 구조

```
lecture/
├── index.html          # 메인 페이지
├── logo.PNG           # 로고 이미지
├── videos/            # 강의 영상 파일 폴더
│   ├── lecture1.mp4
│   ├── lecture2.mp4
│   └── ...
└── README.md          # 이 파일
```

## 사용 방법

### 1. 강의 영상 추가

1. **관리자 모드 로그인**
   - 페이지 하단의 "관리자 모드" 버튼 클릭
   - 기본 비밀번호: `admin`
   - 비밀번호는 `index.html` 파일의 `ADMIN_PASSWORD` 변수에서 변경 가능

2. **영상 파일 업로드**
   - `videos/` 폴더에 영상 파일을 업로드
   - 지원 형식: MP4, WebM, OGG

3. **강의 추가**
   - "강의 추가" 버튼 클릭
   - 강의 제목 입력
   - 비디오 경로 입력 (예: `videos/lecture1.mp4`)
   - "추가" 버튼 클릭

### 2. 강의 목록 관리

- **내보내기**: 현재 강의 목록을 JSON 파일로 다운로드
- **불러오기**: JSON 파일을 선택하여 강의 목록 불러오기
- **삭제**: 관리자 모드에서 각 강의 항목에 마우스를 올리면 "삭제" 버튼 표시

## 데이터 포맷

### 강의 목록 JSON 형식

강의 목록은 다음과 같은 JSON 배열 형식으로 저장됩니다:

```json
[
  {
    "title": "강의 제목",
    "path": "videos/lecture1.mp4"
  },
  {
    "title": "다른 강의 제목",
    "path": "videos/lecture2.mp4"
  }
]
```

### 필드 설명

- **title**: 강의 제목 (문자열)
- **path**: 비디오 파일 경로 (문자열, `videos/` 폴더 기준)

## 비디오 파일 포맷

### 지원 형식

- **MP4** (권장): 가장 널리 사용되는 형식, 모든 브라우저 지원
- **WebM**: 오픈 소스 형식, Chrome, Firefox 지원
- **OGG**: 오픈 소스 형식, Firefox, Chrome 지원

### 권장 사양

- **코덱**: H.264 (MP4의 경우)
- **해상도**: 1920x1080 (Full HD) 또는 1280x720 (HD)
- **비트레이트**: 2-5 Mbps

## 관리자 비밀번호 변경

`index.html` 파일을 열어 다음 부분을 수정하세요:

```javascript
const ADMIN_PASSWORD = 'admin';  // 원하는 비밀번호로 변경
```

## GitHub Pages 배포

1. GitHub 저장소 생성
2. 모든 파일 업로드
3. Settings > Pages에서 GitHub Pages 활성화
4. `videos/` 폴더에 영상 파일 업로드

## 주의사항

- 강의 목록은 브라우저의 로컬 스토리지에 저장됩니다
- 다른 컴퓨터에서 사용하려면 "내보내기"로 JSON 파일을 저장하고 "불러오기"로 불러오세요
- 관리자 모드는 세션 동안만 유지됩니다 (브라우저를 닫으면 로그아웃)
- 비디오 파일은 `videos/` 폴더에 저장해야 합니다
